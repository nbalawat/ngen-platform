Platform & Custom Agents

Visual Workflow Builder

Drag-and-drop canvas with React Flow. Mix sequential, parallel, and graph topologies. AI-assisted generation from natural language descriptions.

🔧

Real Tool Invocation (MCP)

Built-in web search, knowledge base, and document intelligence. Tools execute real operations via Model Context Protocol JSON-RPC.

📚

Agentic RAG Pipeline

Upload PDFs, DOCX, and text files. Automatic parsing, chunking, embedding, and vector search. Tenant-scoped document collections.

💾

7-Type Memory System

Conversational, entity extraction, summarization, tool logs, workflow state, toolbox registry, and knowledge base. All tenant-isolated.

🔒

Multi-Tenant Isolation

Strict tenant scoping across agents, memory, documents, and workflows. Platform agents are read-only. RBAC enforced at every layer.
